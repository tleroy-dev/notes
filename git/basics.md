# Git must known 
This file summarized most used command of git.

## git init
Will init a **new git repo** inside the current folder
```
git init
```

## git add
Add files to the **staging area**
```
git add /my/path/to/file
git add .
```

## git commit
Will create a **new commit** with the file staged.
```
git commit
git commit -m "My commit message"
```

## git push
Transfer local change to **remote repository** (*origin*)
